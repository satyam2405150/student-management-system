# Student Management System

This is a backend project built using Java and SQL to manage student records efficiently.

## Features
- Add new student
- View student details
- Update student information
- Delete student records
- Search students by name or ID

## Tech Stack
- Java
- SQL
- JDBC (for database connectivity)

## Project Objective
- Understand backend logic and database interaction
- Apply Object-Oriented Programming concepts
- Build a structured and scalable system

## Future Improvements
- User authentication (Admin/Login system)
- GUI interface
- Advanced search and filtering

## Status
Currently in development.
