# Student Management System

A full-featured desktop application built using Java, JDBC, MySQL, and Swing to manage students, courses, enrollments, and attendance.

---

## Features

- Student CRUD (Add, Update, Delete, Search)
- Course Management
- Enrollment System (Many-to-Many relationship)
- Attendance Tracking
- Attendance Percentage Calculation
- Filter attendance by student
- Interactive GUI using Java Swing (JTable, Forms, Dropdowns)

---

## Tech Stack

- Java (Core + OOP)
- JDBC (Database Connectivity)
- MySQL (Relational Database)
- Java Swing (GUI)

---

## Concepts Used

- DAO Design Pattern
- SQL JOIN Queries
- Foreign Key Constraints
- Exception Handling
- Separation of UI and Database Logic

---

## How to Run

1. Install MySQL
2. Create database `student_db`
3. Create tables (students, courses, enrollments, attendance)
4. Add MySQL Connector JAR
5. Run `DashboardUI.java`
